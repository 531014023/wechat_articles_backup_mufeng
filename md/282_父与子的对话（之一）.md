# 父与子的对话（之一）

昨日回京路上，与孩子（11岁）有一番对话，很有趣，记录一下。

**子**：爸爸，为什么Nike和Adidas的运动鞋要比李宁、安踏的贵呢？李宁比较好的运动鞋卖2,000元左右，可是Nike、Adidas很多普通的鞋都要卖到这个价格？

**父**：我们以前聊过，价格是由什么决定的呢？

**子：**是由买得起的人决定的。

**父**：是由需求决定的。

**子：**对，需求决定的。

**父**：那你想想，为什么Nike、Adidas的鞋比李宁、安踏的贵呢？

**子：**因为质量更好，想买的人更多？好像也不是。是因为什么呢？

**父**：因为面对的客户不同。Nike、Adidas的品牌时间很长，他们面向的是全球客户，一开始是欧美客户，欧美客户更有钱，所以它们的价格一开始是根据欧美客户的收入来定的。李宁和安踏是国产品牌，主要面对的是中国客户，它们的价格主要是根据中国客户的收入来定的。十几年前，国产品牌运动鞋也就卖200、300元，这几年价格提高到了千元以上。所以产品要想卖得更贵，要卖给谁呢？

**子：**要卖给更有钱的人。

**父：**对，要卖给更有钱的人。

**子：**那国产运动鞋为什么不卖给国外更有钱的人呢？

**父**：这要看国外消费者是否接受中国品牌的运动鞋。如果中国品牌的产品只卖给中国客户，那中国人的收入水平就决定了这些产品的价格。但如果要卖到海外去，那就得先让外国人认可这些品牌，需要做很多工作。

**子：**比如？

**父**：比如要做品牌运营，要打广告，要有文化宣传。比如赞助大型体育赛事，让乔丹、C罗这些巨星也穿上。这要花很多钱。又比如安踏收购了FILA在中国大陆和港澳的运营权，FILA是一个欧洲品牌，它的定价原本就是根据欧美市场来确定的，所以目前价格还是比国产品牌要高。

**子：**那为什么欧美的人更有钱呢？

**父**：这个问题很好，也很复杂。一个主要原因是，欧美的企业更多并且更有钱，他们的员工也就更有钱。你知道一个普通中国人的月收入是多少吗？

**子：**不知道。

**父**：我们在国内旅游时，很多地方餐厅里的服务员，月收入大概在3,000-4,000元人民币，快递、外卖员月收入大概在5,000元人民币，如果是在北京上海这些大城市，他们的收入会再高一些，但不太会超过每月1万元人民币。那你算算，中国餐厅里的服务员，一年能挣多少钱？

**子：**一年3万多4万元人民币。

**父：**那你知道美国人的收入情况是怎样吗？

**子：**不知道。

**父**：美国的联邦贫困线（除两个州外），单人家庭是每年约1.5万美元，也就是，如果一年收入不到1.5万美元，在美国就是贫困人口，政府就要在医疗和食品上进行补贴，虽然过不好，但也不会挨饿。美元兑人民币目前大概是1:7，你算算1.5万美元是多少人民币？

**子：**差不多10万元人民币。

**父：**对，在美国一年挣10万人民币，只是刚刚脱贫。

**子：**那确实是美国人更有钱。怎么才能让中国人更有钱呢？

**父：**这是个好问题，你有什么办法？

**子：**可以给大家多发钱！

**父：**怎么发呢？

**子：**每个人都发一样的钱，这样大家就都有钱了。

**父：**那如果每个人的收入都是一样的，接下来会发生什么呢？

**子：**会有人反抗。

**父：**啊？为什么？

**子：**因为能干的人不满意。

**父：**为什么不满意？

**子：**因为能干的人得不到更多。

**父：**那假设，能干的人也不反抗，会怎样呢？

**子：**那能干的人就会走。

**父：**那如果也走不了呢？

**子：**那大家就都不干了。

**父：**就都躺平了。为什么会这样？

**子：**因为不公平。

**父：**就像以前M老师当你们班主任的时候？

**子：**对，M老师当班主任的时候，只对少数同学好，其他能干的同学得不到表扬，没有机会，不公平，大家就没有动力了。

**父：**你讲得很好。五十年前中国大陆搞过计划经济，那时候就是这种情况。产品的价格是定死的，每个人的工资也是定死的，干与不干都一样，不公平，最后经济就出了大问题，老百姓变得非常穷。

**子：**那个时候能干的人都不干了，那谁会升职呢？

**父：**你觉得呢？

**子：**靠关系，亲戚朋友，不公平。

**父：**对，就是这样。1978年之后才有转变，邓小平领导改革开放，搞市场经济，发展了四十多年，你才能看到今天。

**子：**什么是市场经济？

**父：**举个简单例子，一个人种橙子卖，在计划经济的时候，他种的数量、卖的价格都是定死的，想多种、多卖都不行。后来放开了，他种的橙子好吃，有人愿意多买，那他就可以多种、多卖，这样他的收入就提高了。他挣的钱多起来，就会买货车、建仓库，这样车也能多卖。每个人都这样，经济就好起来了，大家就都挣钱了。

**子：**明白了。那一个企业怎么才能挣更多的钱呢？

**父：**怎样才代表一个企业挣更多的钱呢？

**子：**看利润率。

**父：**你怎么知道这个概念的？

**子：**你以前好像跟我讲过。

**父：**好吧。那利润是怎么算出来的？

**子：**卖出去的价格减去本来要花的钱。

**父：**对，收入减去成本。那你说怎样才可以提高利润呢？

**子：**卖更高的价格。

**父：**但有可能客户承受不起那么高的价格，反而卖得更少。

**子：**还有就是降低成本。

**父：**如何降低企业的成本呢？

**子：**减税。

**父：**你知道怎么这个的？

**子：**以前好像也聊过。

**父：**好吧。那除了减税之外呢？企业的成本除了税收之外，还包括哪些？

**子：**我想想。还有设备，人工。

**父：**对的，还有研发投入、广告支出等等。

**子：**研发投入不就是人工吗？

**父：**不是，研发投入是形成技术或者知识产权的，另外计算。

**子：**知识产权是什么？

**父：**比如，写一本书，会有著作权，如果出版社把书卖掉，就会给一笔钱给作者，这就是知识产权收入。又比如，你玩的游戏，塞尔达、原神，都是软件，不是实物，都算知识产权。还有手机、电脑里的芯片，虽然是实物，但它们大多都是申请了专利的产品，有专利权，也是知识产权。还有你喜欢的画画，车的外观设计，也是知识产权。知识产权是脑力的创造，是可以用来赚大钱的。

**子：**那塞尔达更赚钱还是原神更赚钱？

**父：**你觉得呢？

**子：**原神更赚钱。

**父：**为什么？

**子：**原神抽卡每次都要花钱，塞尔达不用抽卡，只花一次钱。

**父：**这就是游戏的两种收费模式，确实原神更赚钱。

**子：**原神抄塞尔达的。

**父：**模仿痕迹很重，但也有创新。

**子：**还是不明白为什么美国人更有钱。我看新闻里说美国经常有校园枪击案，很危险。

**父：**美国允许老百姓持枪，持枪问题确实是美国特有的一个问题，美国内部也有很多争议。但其实每个国家都有自己的问题，你要想了解一个国家，不能只看中文的信息。中文关于美国的信息可能是不全面的，有选择的报道。

**子：**为什么要选择？

**父：**有可能是因为最近关系不太好。美国新闻媒体也会过滤来自中国的信息，报道中国不好的事。要了解真实全面的情况就不能只看一部分媒体。所以学习英语很重要，因为你已经会中文了，看中文信息没有问题。但英语是更加通用的世界语言，通过不同的英文媒体你可以了解来自世界各国的信息，比只看中文信息要全面。而且全球最优秀的研究机构和学者仍然是集中在美国和欧洲，其中也有很多优秀的华人学者，很多顶尖研究（包括AI）都是基于英语来做的。

**子：**好吧。为什么他们都到欧美去了？

**父：**原因有很多，可能因为觉得学术研究更自由，不受限制；学术评价更加专业，不是靠。。。

**子：**不是靠关系。

**父：**更看重个人真实的能力。所以在这种环境中做研究更多是出于自己的兴趣，而不是谋一份工作，尤其对顶尖的研究人员来讲。就像你喜欢画画一样，如果是每天有任务，有各种标准，你还愿意画吗？

**子：**不愿意，没意思。

**父：**是这样。此外，这些研究机构给优秀学者的收入都不低。

**子：**哪里来的钱呢？

**父：**公立高校是靠政府给钱，私立高校是靠捐赠，捐赠后成立学校的基金，用于投资，再加上学费收入。

**子：**那我觉得他们那里还行。

**父：**行与不行，要根据自己的需求来判断，别人的判断不重要，每个人的需求都不同。

**子：**看起来中国还很落后。

**父：**也不能这么理解。中国比大多数国家发展得都要好。很多国家，包括非洲、南亚的大国，首都都还没有自来水，高铁、4G网络都没有。但比欧美最发达的国家，中国确实还有很多地方有很大差距。

**子：**中国的电动车也不错，是不是比欧美要更好？

**父：**TESLA很好，中国的电动车目前也非常好。但TESLA现在已经是全球品牌，BYD的海外销量也不错，其他国产品牌的电动车整体销量还很少，海外销量更少，能销往发达国家赚钱的就更少了。

**子：**还是要多挣有钱人的钱。

**父：**对。实际上最近两年有些国外品牌的汽车，在中国市场销量很差，但在海外盈利很好，比如欧洲的Stellantis，就是生产标致、雪铁龙的那家，利润都创新高了。

**子：**好了，先不说了，我要继续构思我的游戏了。

**父：**好吧。

以上。

# 太阳照常升起 - 文章存档

本仓库包含微信公众号「太阳照常升起」的历史文章存档。

## 目录结构

```
太阳照常升起/
├── html/              # 可见HTML文件（用于阅读）
├── html_source/       # HTML缓存文件（原始抓取内容）
├── md/                # Markdown格式文章（推荐阅读）
├── fetch.log          # 抓取日志（已忽略）
└── progress.json      # 抓取进度（已忽略）
```

## 文件夹说明

### 📄 html/
包含提取后的可见HTML文件，文件名格式：`{序号}_{文章标题}_visible.html`

- 这些文件经过提取处理，只保留文章内容部分
- 适合在浏览器中直接打开阅读
- 保留了原文的排版和图片

### 💾 html_source/
包含原始HTML缓存文件，文件名格式：`{序号}.cache.html`

- 这些是抓取时的原始HTML完整页面
- 用于断点续传和重新提取
- 包含完整的微信页面结构

### 📝 md/
包含转换后的Markdown格式文件，文件名格式：`{序号}_{文章标题}.md`

- 推荐用于阅读和处理
- 纯文本格式，便于搜索和编辑
- 保留了文章链接和图片引用

## 文件命名规则

- **序号**: 文章编号（如 392、391...）
- **文章标题**: 原始文章标题（经过文件名安全处理）
- **后缀**: 
  - `.cache.html` - 原始缓存
  - `_visible.html` - 可见HTML
  - `.md` - Markdown格式

## 数据来源

- 公众号: 太阳照常升起
- 抓取时间: 2026年
- 文章总数: 437篇（持续更新）

## 使用建议

1. **阅读**: 推荐使用 `md/` 文件夹下的 Markdown 文件
2. **查看原始排版**: 使用 `html/` 文件夹下的可见HTML文件
3. **重新处理**: 如需重新提取，可使用 `html_source/` 下的缓存文件

## 忽略文件

以下文件不会被上传到GitHub：
- `fetch.log` - 抓取过程日志
- `progress.json` - 抓取进度状态
